# Gate UI 界面布局与交互说明

主题: 深色科技风  `#0f1117` / accent `#4f8ef7` / 字体: 思源黑体

---

## 一、主窗口

```
+----------+-----------------------------------+
| [G] 全局代理  <- 选中:蓝字+左竖条 3px      |
| [A] 应用代理                               |
| [P] 预  设                                |
| [S] 状态总览                               |
| [T] 连通测试                               |
| ─────                                     |
| [W] 配置向导  <- 紫色                       |
+----------+-----------------------------------+
```

导航：默认灰 → hover `#1e2436` → 选中 `#1e2d4f`+蓝字+左边框 3px，切换调用 `Refresh()`

---

## 二、全局代理 GlobalPanel

```
[状态卡] HTTP_PROXY / HTTPS_PROXY / NO_PROXY 当前值
[表单卡] 代理地址/HTTP/HTTPS/NO_PROXY 输入框 + 测试 toggle
[操作行] 清除代理(红) | 刷新(次要) | 应用设置(主要)
[反馈标签]
```

- 应用设置 → 状态卡变黄色，反馈绿色
- 清除代理 → 状态卡灰色斜体
- 验证失败 → 反馈红色

---

## 三、应用代理 AppPanel

```
[工具栏] 搜索框 | 分类下拉 | 仅已安装 toggle
[列表]   应用名 | 分类 | 状态点 | 设置/清除按钮  (每行 44px)
[批量栏] 已选N个 | 批量代理框 | 全选已安装 | 批量设置 | 批量清除
```

状态点：绿 `#22c55e` 已配置 / 灰 `#334155` 未配置

弹窗 edit-overlay（默认 hidden）：点击「设置」触发，保存后刷新状态点。

---

## 四、预设 PresetPanel

```
[左列 210px] 预设列表 + [+新建]
[右列]       名称/时间/代理值(黄)/应用数 + 应用/设默认/删除
```

new-preset-overlay：输入名称+描述，创建后刷新列表。

---

## 五、状态总览 StatusPanel

```
[全局代理卡] HTTP/HTTPS/NO_PROXY + [编辑] + [刷新]
[应用代理卡] 已配置N个 + 分类分组工具列表（绿点+名称+代理值）
[当前预设卡] 预设名称蓝色粗体
```

---

## 六、连通测试 TestPanel

```
[表单] 代理框 + [使用当前] / 测试URL框
[大按钮] 开始测试
[结果] success卡(OK+延迟+URL) / fail卡(ERR+错误信息)  默认hidden
[历史] ListView max-height 180px
```

---

## 七、控件样式

| 控件 | 关键样式 |
|------|----------|
| TextField | 背景 `#0a0e18` 边框 `#1e293b` 圆角 6px 高 34px；focus 边框 `#4f8ef7` |
| Toggle | 18x18 圆角 4px；选中 `#4f8ef7` |
| Dropdown | 同 TextField；下拉菜单 `#111827` 圆角 8px |
| ListView | 背景 `#0a0e18`；hover `#131f35`；selected `#1e2d4f`；滚动条 8px |
| btn-primary | `#4f8ef7` 白字 hover `#6ba3f9` |
| btn-secondary | `#1e293b` 灰字 hover 边框蓝 |
| btn-danger | 透明背景 红字 `#f87171` 红边框 |

---

## 八、弹窗通用规则

- 默认 `display:none`，C# 用 `DisplayStyle.Flex` 显示
- 遮罩：`position:absolute` 全面板覆盖 `rgba(0,0,0,0.75)`
- 面板：宽 460px 背景 `#111827` 圆角 14px padding 28×32px
- 按钮组：右对齐，取消在左（次要），确认在右（主要）
