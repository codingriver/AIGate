## 4.1 ProxyConfig

```csharp
public class ProxyConfig {
    public string? HttpProxy  { get; set; }  // http:// 开头，含端口，如 http://127.0.0.1:7890
    public string? HttpsProxy { get; set; }  // 通常与 HttpProxy 相同
    public string? NoProxy    { get; set; }  // 逗号分隔，支持通配符，如 "localhost,127.0.0.1,*.local"
    public bool IsEmpty => HttpProxy == null && HttpsProxy == null && NoProxy == null;
}
```

**代理地址格式约束**：
- 标准格式：`http://host:port` 或 `http://user:pass@host:port`
- 支持协议：`http://`、`https://`、`socks5://`（仅环境变量层，不写入工具配置文件）
- 不含尾部斜杠
- 端口范围：1–65535

## 4.2 Profile（预设）

```csharp
public class Profile {
    public string Name        { get; set; }  // 预设名，唯一键，用于文件名 {name}.json
    public string Description { get; set; }  // 可为空字符串
    public DateTime CreatedAt { get; set; }  // UTC 时间
    public DateTime UpdatedAt { get; set; }  // 每次 preset save 时更新
    public ProxyConfig EnvVars { get; set; }             // 全局环境变量代理快照
    public Dictionary<string, ProxyConfig> ToolConfigs { get; set; }  // key = toolName
}
```

**存储路径**：`{DataDir}/profiles/{name}.json`

**命名约束**：
- 只允许字母、数字、连字符、下划线
- 最大长度 64 字符
- 不区分大小写（文件系统兼容）
- 保留名：`default`（作为未命名预设的占位）

**文件格式示例**：
```json
{
  "name": "office",
  "description": "公司内网代理",
  "createdAt": "2026-01-01T08:00:00Z",
  "updatedAt": "2026-03-17T22:00:00Z",
  "envVars": {
    "httpProxy": "http://proxy.company.com:8080",
    "httpsProxy": "http://proxy.company.com:8080",
    "noProxy": "localhost,127.0.0.1,*.company.com"
  },
  "toolConfigs": {
    "git":  { "httpProxy": "http://proxy.company.com:8080", "httpsProxy": null, "noProxy": null },
    "npm":  { "httpProxy": "http://proxy.company.com:8080", "httpsProxy": null, "noProxy": null }
  }
}
```

## 4.3 GlobalConfig（全局配置）

```csharp
public class GlobalConfig {
    public string? DefaultPreset   { get; set; }  // gate preset set-default 设置
    public string? DefaultTestUrl  { get; set; }  // gate test 默认测试目标，默认 https://www.google.com
    public int     TestTimeoutMs   { get; set; }  // 默认 10000ms
    public bool    AutoSaveHistory { get; set; }  // 默认 true，gate set 时自动记录历史
    public int     MaxHistoryCount { get; set; }  // 默认 20
    public string? ProxyExpiresAt  { get; set; }  // ISO8601，gate set --expires 写入，过期后自动 clear（P1）
    public string? UiTheme         { get; set; }  // "dark" | "light"，Unity GUI 主题（P2）
}
```

**`DefaultTestUrl` 说明**：`gate test` 无参数时使用此 URL 测试；可通过 `gate config set defaultTestUrl https://...` 修改（P1 提供配置命令）。默认值 `https://www.google.com` 在中国大陆可用于验证代理是否能访问境外服务。

**存储路径**：`{DataDir}/config.json`

## 4.4 ToolDescriptor（声明式工具 JSON）

### 完整字段定义

```csharp
public class ToolDescriptor {
    public string ToolName    { get; set; }  // 必填，小写+连字符，唯一标识
    public string DisplayName { get; set; }  // 可选，展示名，默认同 ToolName
    public string Category    { get; set; }  // 可选，分类名
    public ToolConfigDescriptor? Config   { get; set; }  // 配置文件型，与 EnvProxy 二选一
    public EnvProxyDescriptor?  EnvProxy  { get; set; }  // 环境变量型
}

public class ToolConfigDescriptor {
    public List<PlatformString> ProbePaths  { get; set; }  // 配置文件探测路径列表（按序尝试）
    public string Format      { get; set; }  // keyvalue | ini | json | toml | xml
    public List<ProxyLine>    ProxyLines  { get; set; }  // 要写入的代理行
    public string? Section    { get; set; }  // ini/toml 格式的 section 名，如 "http"
}

public class ProxyLine {
    public string Key   { get; set; }  // 配置文件键名
    public string Value { get; set; }  // 值模板，{proxy} 替换为实际代理地址
    public bool   RemoveOnClear { get; set; }  // 默认 true，clear 时删除此行
}

public class EnvProxyDescriptor {
    public List<string> EnvVars { get; set; }  // 要设置的环境变量名列表
}
```

### 配置文件型示例（npm）
```json
{
  "toolName": "npm",
  "displayName": "npm",
  "category": "包管理器",
  "config": {
    "probePaths": [
      { "windows": "%USERPROFILE%\\.npmrc", "linux": "~/.npmrc", "macos": "~/.npmrc" }
    ],
    "format": "keyvalue",
    "proxyLines": [
      { "key": "proxy",       "value": "{proxy}", "removeOnClear": true },
      { "key": "https-proxy", "value": "{proxy}", "removeOnClear": true }
    ]
  }
}
```

### 环境变量型示例（openai）
```json
{
  "toolName": "openai",
  "displayName": "OpenAI",
  "category": "AI 云服务",
  "envProxy": {
    "envVars": ["OPENAI_PROXY", "HTTP_PROXY"]
  }
}
```

### PlatformString 两种写法

| 写法 | 适用场景 | 示例 |
|------|----------|------|
| 统一字符串 | 三平台路径一致 | `"~/.npmrc"` |
| 差异化对象 | 三平台路径不同 | `{ "windows": "%APPDATA%\\npm\\.npmrc", "linux": "~/.npmrc", "macos": "~/.npmrc" }` |

`~` 在所有平台自动展开为用户主目录；`%VAR%` 仅在 Windows 展开。

## 4.6 PluginInfo（插件元数据）

```csharp
public class PluginInfo {
    public string   ToolName    { get; set; }
    public string   SourceFile  { get; set; }  // 安装时的原始文件路径（本地安装为绝对路径，社区安装为 downloadUrl）
    public DateTime InstalledAt { get; set; }  // UTC
    public string   InstalledBy { get; set; }  // "local" 或 "community"
    public string   Version     { get; set; }  // 取自 tool.json 的 version 字段；未声明时为 ""
    public string   Sha256      { get; set; }  // 社区安装时记录校验值；本地安装为 ""
}
```

元数据存储于 `{DataDir}/plugins/{toolName}/plugin-info.json`。

## 4.7 BackupInfo（备份元数据，P1）

```csharp
public class BackupInfo {
    public string   Timestamp   { get; set; }  // "20260317-220000"，同时作为目录名
    public DateTime CreatedAt   { get; set; }  // UTC
    public List<BackupEntry> Entries { get; set; }
}

public class BackupEntry {
    public string ToolName      { get; set; }  // 工具名
    public string OriginalPath  { get; set; }  // 原始配置文件绝对路径
    public string BackupFile    { get; set; }  // 备份目录内的相对文件名
}
```

存储路径：`{DataDir}/backups/{timestamp}/backup-info.json`  
备份文件：`{DataDir}/backups/{timestamp}/{toolName}-config`（原始文件内容，无后缀）

## 4.8 存储路径（GatePaths）

| 条目 | Windows | Linux | macOS |
|------|---------|-------|-------|
| 数据根 | `%LOCALAPPDATA%\gate` | `~/.local/share/gate` | `~/Library/Application Support/gate` |
| 预设 | `{DataDir}\profiles\` | `{DataDir}/profiles/` | 同左 |
| 插件 | `{DataDir}\plugins\` | `{DataDir}/plugins/` | 同左 |
| 历史 | `{DataDir}\history.json` | 同左 | 同左 |
| 工具路径 | `{DataDir}\tool-paths.json` | 同左 | 同左 |
| 全局配置 | `{DataDir}\config.json` | 同左 | 同左 |
| 备份（P1） | `{DataDir}\backups\` | `{DataDir}/backups/` | 同左 |

**Linux XDG 说明**：`~/.local/share/gate` 符合 XDG Base Directory 规范（`$XDG_DATA_HOME/gate`）。若 `$XDG_DATA_HOME` 已设置则优先使用该变量。
