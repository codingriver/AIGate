## 1. 功能概述

插件系统允许用户安装第三方工具定义（JSON 格式），扩展 Gate 支持的工具列表。插件可覆盖同名内置工具，支持热注册（无需重启），安装后立即可用。

---

## 2. 插件文件格式

```json
{
  "name": "my-tool",
  "displayName": "My Custom Tool",
  "category": "自定义",
  "description": "自定义代理工具",
  "config": {
    "format": "keyvalue",
    "configFiles": [
      { "path": "~/.my-tool/config", "platform": "all" }
    ],
    "proxyKey": "proxy",
    "httpsProxyKey": "https-proxy"
  },
  "probePaths": [
    { "path": "~/.my-tool", "platform": "all" }
  ]
}
```

**必填字段**：`name`、`config.format`、`config.configFiles`（≥1 个）  
**可选字段**：`displayName`、`category`、`description`、`probePaths`  
**`format` 合法值**：`keyvalue`、`ini`、`json`、`toml`

## 3. 主要功能

### 3.1 列出已安装插件

**命令**：`gate plugin list`

表格输出：名称、来源文件路径、是否覆盖内置、安装时间。

### 3.2 本地安装插件

**命令**：`gate plugin install path/to/tool.json`

1. 读取并校验 JSON schema
2. 通过后复制到 `{DataDir}/plugins/{name}/tool.json`
3. 热注册到 `ToolRegistry`
4. 若同名**内置工具**存在：**必须加 `--force` 才能覆盖，否则报错退出**；同时输出 WARN 提示用户内置工具将被覆盖（A11：方案B + 提示）
5. 失败时清理残余文件

### 3.3 卸载插件

**命令**：`gate plugin remove <name>`

- 删除 `{DataDir}/plugins/{name}/` 目录
- 从 `ToolRegistry` 热注销
- 若有同名内置工具，自动恢复内置版本

### 3.4 校验插件格式

**命令**：`gate plugin validate path/to/tool.json`

- 只做 schema 校验，不安装
- 输出所有字段错误列表
- 退出码：`0` 合法，`1` 有错误

### 3.5 查看插件详情

**命令**：`gate plugin show <name>`

- 格式化输出插件 JSON，显示来源路径和安装时间

### 3.6 热注册机制

安装/卸载后立即调用 `ToolRegistry.Reload()`，无需重启 Gate 进程。

### 3.7 社区索引（P1）

- `gate plugin search <kw>`：从 `GlobalConfig.PluginRegistryUrl` 拉取索引，关键词模糊匹配名称/描述
- `gate plugin install <id>`：下载并安装社区插件
- `gate plugin update <name>/--all`：更新已安装社区插件

## 4. 技术实现要点

```csharp
// Gate.Core/Plugin/PluginManager.cs
public class PluginManager {
    public List<PluginInfo> ListAll();             // 扫描 {DataDir}/plugins/
    public InstallResult Install(string filePath); // 校验 + 复制 + 热注册
    public void Remove(string name);               // 删除目录 + 热注销
    public ValidationResult Validate(string filePath); // 仅校验
    public PluginInfo? Get(string name);           // 读取详情
}

public record PluginInfo(
    string Name,
    string FilePath,
    bool OverridesBuiltin,
    DateTime InstalledAt
);

public record InstallResult(bool Success, string? ErrorMessage);
public record ValidationResult(bool IsValid, IReadOnlyList<string> Errors);
```

**工具加载优先级**：
```
内置 EmbeddedResource JSON  (最低)
→ 特殊 C# Configurator     (覆盖同名内置)
→ 用户插件 plugins/**      (最高，覆盖一切)
```

## 5. 异常处理

| 异常场景 | 行为 | 退出码 |
|----------|------|--------|
| 文件不存在 | 参数错误 | 1 |
| JSON 解析失败 | 输出解析错误行号 | 1 |
| schema 校验失败 | 列出所有字段错误 | 1 |
| `name` 已被**内置工具**占用 | 必须加 `--force` 才能覆盖，同时输出 WARN 提示（A11） | 2 |
| `name` 已被**其他插件**占用 | 提示冲突，`--force` 覆盖 | 2 |
| plugins/ 目录不可写 | 输出错误 | 4 |
| 卸载不存在的插件 | 输出错误 | 2 |

## 6. 验收标准

- [ ] `gate plugin install tool.json` 安装后，`gate app my-tool` 可用
- [ ] 安装与内置工具同名的插件时，无 `--force` 报错，加 `--force` 后安装成功并输出 WARN（A11）
- [ ] 安装覆盖同名内置工具，卸载后内置工具恢复
- [ ] `gate plugin validate` 对缺少必填字段的 JSON 返回退出码 1 和具体错误
- [ ] 安装失败时 `{DataDir}/plugins/` 中无残余文件
- [ ] 热注册：安装后无需重启即可使用
