## 注意事项

1. **特殊 Configurator**：带「C# Configurator」标注的工具需要专用代码实现（非纯 JSON 声明）
2. **环境变量格式**：标注「环境变量」的工具通过设置系统环境变量而非修改配置文件来配置代理
3. **平台差异**：配置文件路径在 Windows/macOS/Linux 上可能不同，`ToolDescriptor.Config.ConfigFiles` 的 `platform` 字段控制平台过滤
4. **工具检测**：`probePaths` 为空的工具通过 PATH 中是否存在可执行文件来判断是否安装
5. **插件扩展**：用户可通过 `gate plugin install` 添加本文档未列出的工具
