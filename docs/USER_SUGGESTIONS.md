# Gate 用户体验建议

> 从用户角度出发的改进建议与功能补充

---

## 一、核心体验问题

### 1.1 首次使用门槛高

**问题**：用户拿到工具后不知道从哪里开始，工具名称和分类不够直观。

**建议**：
- 启动后自动检测系统已安装的工具，弹出「快速配置」引导
- 一键检测按钮：扫描系统工具并显示哪些已配置、哪些未配置
- 欢迎界面显示「3步快速开始」

### 1.2 代理状态不够直观

**问题**：用户不清楚当前代理是否生效，工具配置状态分散。

**建议**：
- 顶部状态栏显示当前代理状态（绿色/红色指示灯）
- 一键「测试所有已配置工具」按钮
- 应用面板列表行显示「最后设置时间」

### 1.3 错误信息不友好

**问题**：错误时只显示英文技术信息，用户不知道如何处理。

**建议**：
- 错误信息本地化（中文）
- 每种错误附带「可能的解决方案」
- 代理连接失败时给出常见原因列表（防火墙/端口/认证）

## 二、功能补充建议

### 2.1 代理地址管理

**痛点**：用户在多个场景使用多个代理地址，每次手动输入很麻烦。

**建议**：
- 「常用代理地址」收藏夹，支持命名（如「公司 VPN」「家用梯子」）
- 输入框支持下拉历史记录（最近使用的10条地址）
- 支持从剪贴板自动识别代理地址格式

### 2.2 代理有效期设置

**痛点**：临时代理忘记清除，影响正常网络访问。

**建议**：
- 设置代理时可选「有效期」（1小时/8小时/直到手动清除）
- 到期自动清除并通知用户
- 系统托盘图标显示代理倒计时

### 2.3 网络环境自动切换

**痛点**：在公司和家里切换网络时需要手动切换预设。

**建议**：
- 检测当前 WiFi SSID，自动应用对应预设
- 检测 VPN 连接状态，自动切换配置
- 规则编辑器：「连接到 Company-WiFi 时自动应用 office 预设」

### 2.4 批量操作增强

**痛点**：批量操作没有预览，不知道哪些工具会被影响。

**建议**：
- 批量操作前显示「影响预览」对话框（共 N 个工具）
- 支持「撤销」操作（Ctrl+Z 回滚上一次批量操作）
- 批量操作结果报告（成功 X 个，失败 Y 个，详情...）

### 2.5 工具路径自定义

**痛点**：用户将工具安装在非标准路径，系统检测不到「已安装」。

**建议**：
- 应用面板「工具路径」列，支持手动指定可执行文件路径
- 路径自定义后持久化保存，下次启动保持
- 「手动指定路径」按钮打开文件选择器
- 路径变更后自动重新检测是否已安装

### 2.6 代理认证支持

**痛点**：企业代理通常需要用户名和密码，现有格式不够明显。

**建议**：
- 代理地址输入框旁增加「用户名」「密码」独立字段
- 密码字段默认遮掩，支持显示/隐藏切换
- 自动格式化为 `http://user:pass@host:port`
- 密码存储使用系统密钥链（Keychain/Credential Manager）

## 三、界面体验建议

### 3.1 应用列表显示优化

**建议**：
- 列表行显示工具 Logo/图标（可选，Community贡献）
- 未安装工具默认折叠，提供「显示全部」开关
- 支持列表/卡片两种视图切换
- 列头支持点击排序（按名称/分类/状态/最后配置时间）

### 3.2 预设管理增强

**建议**：
- 预设支持标签（Tag）便于分组
- 预设卡片显示包含的工具数量和代理地址摘要
- 拖拽调整预设排列顺序
- 预设对比功能（选两个预设查看差异）
- 预设分享：导出为二维码或短链接

### 3.3 深色/浅色主题

**建议**：
- 跟随系统自动切换深色/浅色主题
- 提供多个颜色主题（当前深蓝/高对比/护眼绿）
- 主题设置持久化

### 3.4 多语言支持

**建议**：
- 支持中文/英文切换（界面、错误信息、文档）
- 语言设置持久化
- 社区贡献其他语言翻译

## 四、CLI 体验建议

### 4.1 命令补全

**建议**：
- 提供 bash/zsh/fish/PowerShell 的 Tab 补全脚本
- `gate app -n <TAB>` 自动补全已安装的工具名
- 安装时自动注册补全

### 4.2 输出格式

**建议**：
- `--json` 标志输出结构化 JSON（便于脚本集成）
- `--quiet` 标志只输出错误
- `--no-color` 标志禁用颜色（适合 CI 环境）

### 4.3 配置文件

**建议**：
- 支持 `~/.gaterc` 或 `~/.config/gate/config.yaml` 全局配置
- 项目级 `.gate.yaml`：进入目录自动应用配置
- 配置项：默认代理/默认预设/测试超时/颜色主题

---

## 五、系统集成建议

### 5.1 系统托盘/菜单栏

**建议**：
- macOS：菜单栏图标，显示当前代理状态，点击快速切换预设
- Windows：系统托盘，右键菜单显示预设列表
- Linux：支持 AppIndicator

### 5.2 开机自启

**建议**：
- 可选「开机自启并应用默认预设」
- 支持 Windows 服务/macOS launchd/Linux systemd 注册

### 5.3 浏览器集成

**建议**：
- 浏览器扩展（Chrome/Firefox），与 Gate 同步代理设置
- 支持 PAC 文件生成，供浏览器/系统代理使用
